1. **package.json** : Fichier de configuration du projet avec les métadonnées et les dépendances.

2. **npm (Node Package Manager)** : Gestionnaire de paquets pour installer, mettre à jour et gérer les dépendances.

3. **Modules intégrés (Core Modules)** : Modules internes de Node.js pour des fonctionnalités de base.

4. **Express.js** : Framework web pour créer des applications web et des API.

5. **Routes** : Définition de chemins URL et de fonctions de rappel pour gérer les requêtes.

6. **Middlewares** : Fonctions qui s'exécutent entre les requêtes et les réponses pour des traitements communs.

7. **EJS (Embedded JavaScript)** : Moteur de template pour créer des pages web dynamiques.

8. **MySQL** : Système de gestion de base de données relationnelle.

9. **Fichiers de routes** : Fichiers séparés pour définir les routes de l'application.

10. **Fichiers de vues** : Fichiers avec l'extension .ejs pour créer des pages HTML dynamiques avec EJS.

En combinant ces éléments, vous pouvez développer des applications Node.js performantes et flexibles, avec une gestion de base de données, des pages web dynamiques et des fonctionnalités de routage avancées.
